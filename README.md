# SCUExtension
四川大学教务系统chrome插件

## 功能列表
> 教务系统
>> 两周内免登陆
>>
>> 系统首页绩点计算
>> 
>> 自动填写登录验证码
>>
>> 自动填写选课验证码
>>
>> 一键评教
>>
>> 校历显示优化
>>
> 
>课程中心
>> 修复chrome浏览器下姓名及退出不能点击的bug

大家有什么脑洞请提issue。

本项目已发布在chrome网上应用商店，搜索[**SCUExtension**](https://chrome.google.com/webstore/detail/scuextension/ljmkgohcdjeafplbnncbpekoomklkmen)即可。

### 0.0.23

### 0.0.21

优化模型加载逻辑。
